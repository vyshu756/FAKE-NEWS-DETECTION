🚀 How to Share Your App

3 Easy Ways to Let Anyone Use Your Fake News Detector

⚡ Method 1: Instant Share (Ngrok) 5 MINUTES FREE

Best for: Quick demos, testing, showing to friends

1

Download Ngrok

Visit: https://ngrok.com/download

Extract the file and you're ready!

2

Start Your Backend

python main.py

Keep this terminal running

3

Create Public Tunnel

Open a new terminal and run:

ngrok http 8000

You'll see a URL like:

https://abc123.ngrok.io
4

Share the Link!

Send this to anyone:

https://abc123.ngrok.io/docs

They can now use your API immediately!

🌐 Method 2: Permanent Deployment 30 MINUTES FREE FOREVER

Best for: Portfolio, production use, permanent access

1

Push to GitHub

git init
git add .
git commit -m "Initial commit"
git push origin main
2

Deploy Backend to Render

1. Go to render.com

2. New Web Service → Connect GitHub

3. Select your repository

4. Click Deploy (automatic!)

5. Get URL: https://your-app.onrender.com

3

Deploy Frontend to Vercel

1. Go to vercel.com

2. Import GitHub repository

3. Set root directory: frontend

4. Add environment variable:

REACT_APP_API_URL=https://your-app.onrender.com

5. Deploy!

4

Share Your App!

https://your-app.vercel.app

✅ Works 24/7

✅ Professional URL

✅ No maintenance needed
