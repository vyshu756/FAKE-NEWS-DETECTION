## ⚡ Fastest Way (5 Minutes)

### Step 1: Install Ngrok
Download from: https://ngrok.com/download

### Step 2: Start Backend
```bash
python main.py
```

### Step 3: Create Public URL

## 🌐 For Permanent Deployment

### Option A: Render + Vercel (100% Free)

**Backend (Render):**
1. Push code to GitHub
2. Go to render.com
3. New Web Service → Connect GitHub
4. Deploy automatically
5. Get URL: `https://your-app.onrender.com`

**Frontend (Vercel):**
1. Go to vercel.com
2. Import GitHub repo
3. Set root directory: `frontend`
4. Add env: `REACT_APP_API_URL=https://your-app.onrender.com`
5. Deploy
6. Get URL: `https://your-app.vercel.app`

**Share:** `https://your-app.vercel.app`
